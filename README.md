# commentcoloring
Status coloring and commenting
        if you must,
try this:
<pre>
&lt;h2&gt;

&#x0060;&#x0060;&#x0060;diff
- red
+ green
! orange
# gray
&#x0060;&#x0060;&#x0060;

&lt;/h2&gt;
</pre>

to render something visually appealing such as:
<h1>

```diff
-! Warning !-
```
</h1>

or

<h2>

```diff
+( Venisons dear isn't it? )+
```

</h2>

or

<h3>

```diff
!~ Dwarf Shortage ~!
```

</h3>

or

<h3>

```diff
#=- Stationary Store Moves -=#
```

</h3>
